#### 题目

给定一个非空整数数组，找到使所有数组元素相等所需的最小移动数，其中每次移动可将选定的一个元素加1或减1。 您可以假设数组的长度最多为10000。

例如:

输入:
[1,2,3]

输出:
2

说明：
只有两个动作是必要的（记得每一步仅可使其中一个元素加1或减1）： 

[1,2,3]  =>  [2,2,3]  =>  [2,2,2]



#### 解题思路

分组+绝对值不等式， 结论就是找到中位数



#### python3代码

```
class Solution:
    def minMoves2(self, nums: List[int]) -> int:
        nums.sort()
        res = 0
        n = len(nums)
        for i in nums:
            res += abs(i- nums[n//2])
        return res
```
